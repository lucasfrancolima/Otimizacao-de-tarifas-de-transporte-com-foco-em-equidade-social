Otimização de tarifas de transporte com foco em equidade social

Este repositório disponibiliza os arquivos utilizados no desenvolvimento da tese de doutorado "Otimização de tarifas de transporte com foco em equidade social", defendida na Universidade Federal de Santa Catarina (UFSC) em 2026.

Os arquivos associados à tese estão disponíveis na seção de Releases deste repositório, especialmente na versão v1.0:

Release v1.0 - Arquivos utilizados na tese

Na release, os materiais estão organizados nos seguintes arquivos compactados:

Modelagem.da.Escolha.-.Optima.zip: códigos e arquivos associados à modelagem da escolha do modo de transporte com dados da pesquisa Optima.
Modelagem.da.Escolha.-.PLAMUS.zip: códigos e arquivos associados à modelagem da escolha do modo de transporte com dados do PLAMUS.
Modelagem.da.Tarifa.-.PLAMUS.zip: códigos e arquivos associados à modelagem tarifária com dados do PLAMUS.

Os arquivos Source code (zip) e Source code (tar.gz) são gerados automaticamente pelo GitHub e correspondem ao conteúdo do repositório no momento da release. Os arquivos principais utilizados na tese são os pacotes compactados listados acima.

Finalidade do repositório
Este repositório foi criado como ponto de acesso aos materiais mencionados na tese, com o objetivo de facilitar a consulta, a verificação e a reprodução dos procedimentos desenvolvidos no trabalho.
